# PsyNove

PsyNove is an online psychiatrist consultation platform for Sri Lanka. It helps users discover and book SLMC-verified psychiatrists in Sinhala, Tamil, or English.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3. Run the app:
   `npm run dev`

Open `http://localhost:3000`.
